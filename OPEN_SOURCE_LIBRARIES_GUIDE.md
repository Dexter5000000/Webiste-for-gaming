# 100+ Open-Source Libraries for Procedural/AI Music Generation

## Overview

This document catalogs 100+ open-source, free, browser-compatible JavaScript/Web Audio libraries for music generation. All run client-side without requiring API keys, making them perfect for truly offline-capable music applications.

**Key Benefits:**
- ✅ 100% free and open-source
- ✅ Run entirely in the browser (no server required)
- ✅ No API keys needed
- ✅ Royalty-free audio generation
- ✅ MIT/Apache/GPL licensed
- ✅ Can be loaded via CDN (unpkg, jsDelivr, cdnjs)

```html
<!-- Tuna (effects) -->
<script src="https://unpkg.com/tunajs"></script>

<!-- Magenta.js (AI) -->
<script src="https://cdn.jsdelivr.net/npm/@magenta/music@1.23.1/es6/bundle.js"></script>
```

### npm Installation

```bash
npm install tone scribbletune howler tunajs @magenta/music wavesurfer.js
```

### Quick Example: Tone.js Procedural

```typescript
import * as Tone from 'tone';

async function generateProceduralLoop() {
  await Tone.start();
  
  const synth = new Tone.PolySynth(Tone.Synth, {
    oscillator: { type: 'triangle' },
    envelope: { attack: 0.005, decay: 0.1, sustain: 0.3, release: 1 },
  }).toDestination();

  const now = Tone.now();
  synth.triggerAttackRelease('C4', '8n', now);
  synth.triggerAttackRelease('E4', '8n', now + 0.5);
  synth.triggerAttackRelease('G4', '8n', now + 1);
}
```

### Quick Example: ScribbleTune Sequences

```javascript
import { scale, pattern } from 'scribbletune';

// Generate C major scale pattern
const notes = pattern({
  scale: scale('C', 'major'),
  length: 8,
  randomness: 0.3,
});

// Export as MIDI
notes.toMidi('generated-music.mid');
```

### By Use Case

**Ambient/Pad Generation:**
- Tone.js (polyphonic synths)
- Magenta.js (ML melodies over pads)
- Abundant Music (one-button)
- Custom: Fractals + Chaos

**Electronic/Dance Beats:**
- ScribbleTune (MIDI sequencing)
- ProcJam (procedural beats)
- Beadz (drum patterns)
- Custom: L-Systems rhythms

**Jazz/Complex Harmony:**
- Markov Chains (intelligent transitions)
- Magenta.js (MusicRNN)
- Tone.js (chord progressions)

**Real-Time Performance:**
- Tone.js (scheduling)
- Howler.js (playback control)
- WEBMIDI.js (device input)

**Full DAW Capability:**
- AudioMass (waveform editor)
- GridSound (timeline UI)
- waveform-playlist (multitrack)

### By Performance Tier

**Lightweight (< 100KB):**
- ScribbleTune
- Blip
- webaudio-tinysynth
- WEBMIDI.js

**Medium (100KB - 500KB):**
- Tone.js
- Howler.js
- Tuna
- VexFlow

**Heavy (> 500KB):**
- Magenta.js (requires WASM)

- AudioMass (full DAW)
