- [ ] Claim Zyte offer

### Deploying Spider
- [ ] Install Scrapy: `pip install scrapy scrapycloud`
- [ ] Create project: `scrapy startproject`
- [ ] Copy template: `freesound-basic.py`
- [ ] Test locally: `scrapy crawl`
- [ ] Deploy: `scrapycloud deploy`

- [ ] Download data: `scrapycloud job`
